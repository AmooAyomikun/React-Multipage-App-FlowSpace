# FlowSpace

A modern multi-page SaaS marketing website built with **React** and **React Router**, showcasing a fictional project management platform called **FlowSpace**.

FlowSpace is designed to demonstrate modern frontend development practices including routing, dynamic pages, API integration, form validation, responsive design, reusable components, and clean UI architecture.

---

## Project Overview

FlowSpace is a professional SaaS landing website that helps teams organize tasks, collaborate efficiently, and streamline project workflows.

The application includes multiple pages connected through React Router and features a fully functional blog system powered by API data.
